# Geometry Tutor 特殊规则

## 现状

Geometry Tutor 当前处于维护阶段。

禁止继续增加新功能。

---

## 优先级

### P0 — 核心稳定性

| 任务 | 说明 |
|------|------|
| SceneIR 收口 | 所有 3D 场景统一走 SceneIR，禁止绕过 |
| Tutor 收口 | 讲解流程统一，消除碎片化 |
| Memory 收口 | 用户学习状态统一管理 |
| Workspace 稳定性 | 状态持久化、回放、导出 |
| 手机端适配 | 390px 宽度无横向滚动、按钮可点击 |

### P1 — 代码清理

| 任务 | 说明 |
|------|------|
| 删除死代码 | 全局扫描未引用组件/函数/路由 |
| Prompt 统一 | 所有 AI Prompt 使用同一体系 |
| 状态管理统一 | 消除重复的状态层 |

### P2 — 新功能（冻结）

| 任务 | 说明 |
|------|------|
| 所有新功能 | 满足解锁条件后才允许 |

---

## 解锁条件

### 必须同时满足

- [ ] 连续 7 天无 P0/P1 Bug
- [ ] SceneIR 收口完成
- [ ] Tutor 收口完成
- [ ] Memory 收口完成
- [ ] 死代码清理完成

---

## 禁止操作

除非经过架构评审，否则禁止：

- 引入新的 3D 渲染方式
- 增加新的讲解模式
- 增加新的题型支持
- 增加新的用户设置
- 重构现有稳定模块

---

## 例外

以下操作不需要解锁：

- 修复 Bug
- 性能优化（不改变架构）
- 删除死代码
- 更新依赖
- 补充测试
- 更新文档
