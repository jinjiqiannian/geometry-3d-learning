# ExamMind 特殊规则

## V1 范围

ExamMind V1 严格遵守以下范围。

---

## 允许的功能

| 功能 | 说明 |
|------|------|
| 试卷上传 | 图片/PDF 上传 |
| OCR | 题目识别与文本化 |
| 错题识别 | 自动标记错题 |
| 学习报告 | 简单统计与报告 |

---

## 禁止的功能

以下功能全部进入 Backlog，V1 版本禁止开发：

| 功能 | 理由 |
|------|------|
| AI Agent | 复杂度高，维护成本大 |
| 知识图谱 | 超出 V1 验证范围 |
| 老师端 | 增加产品复杂度 |
| 学校端 | 增加产品复杂度 |
| 社交系统 | 增加产品复杂度 |
| 推荐系统 | 增加产品复杂度 |
| 复杂分析 | 超出 V1 范围 |
| 题目讲解 | 超出 V1 范围 |
| 学习路径 | 超出 V1 范围 |
| 智能组卷 | 超出 V1 范围 |

---

## 架构约束

- 禁止引入大型 AI 框架
- 禁止引入图数据库
- 禁止引入实时通信
- 禁止引入第三方登录
- OCR 使用现有 AI 能力，禁止自建

---

## Backlog 管理

### 存入 Backlog 的条件

- 属于"禁止的功能"列表中的功能
- 不属于 V1 范围
- 无明确用户验证数据

### 从 Backlog 取出的条件

- V1 验证完成
- 有明确用户需求数据
- 有资源排期
- 通过规格书评审 + 架构评审

---

## 例外

以下操作不需要经过 Backlog 流程：

- OCR 识别率提升（不改架构）
- 试卷上传体验优化（不改流程）
- 学习报告样式调整（不改数据模型）
- 性能优化
- Bug 修复
