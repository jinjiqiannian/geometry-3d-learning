# 几何维度 — 用户增长策略

> 当前阶段: 积累用户和口碑（用户增长 > 用户留存 > 付费转化）

---

## 一、产品定位

**几何维度 = AI立体几何学习工作台**

| 不是 | 而是 |
|------|------|
| AI搜题工具 | AI老师 |
| AI答案生成器 | AI讲解器 |
| 收费解题工具 | AI几何可视化平台 |

---

## 二、增长飞轮

```
用户免费使用 → 解决问题 → 分享给同学 → 新用户进入 → 口碑扩散
                                    ↓
                              教师发现 → 课堂使用 → 批量学生进入
```

**核心假设**: 让一个学生完整解决一道题，比任何广告都有效。

---

## 三、获客渠道

### 3.1 自然搜索 (SEO)

- **长尾关键词**: "正方体体对角线长度"、"异面直线夹角"、"正四棱锥体积"
- **题目页静态生成**: 每个例题独立 URL，可被搜索引擎索引
- **结构化数据**: Schema.org `MathSolver` 标记

### 3.2 分享裂变

- **题目分享链接**: 学生解完题可一键分享给同学
- **3D 场景分享**: 分享带当前视角的 3D 几何场景
- **分享文案优化**: "我用几何维度解了这道题，3D 图形一看就懂"

### 3.3 教师渠道

- **教师备课场景**: 一键生成 PPT 课件
- **课堂演示**: 投影仪优化展示模式
- **教师推荐**: 教师在班级群推荐 → 全班学生使用

### 3.4 内容营销

- **B站/抖音**: 3D 几何可视化短视频（"正方体体对角线，3D 一看就懂"）
- **知乎**: 立体几何学习方法文章
- **小红书**: 学习工具推荐

### 3.5 产品驱动增长 (PLG)

- **无需注册即可使用**: 降低首次使用门槛
- **快速体验入口**: 首页预设例题一键体验
- **渐进式功能披露**: 先体验核心价值，再引导注册/升级

- 连续学习天数
- 学习成就系统

## 五、付费转化

### 5.1 转化时机

| 触发条件 | 转化提示 |
|----------|----------|
| 每日额度用完 | "今日50题已用完，升级无限使用" |
| 多次使用同一题型 | "这个知识点你做了很多题，开启错题本管理" |
| 分享给多人 | "你的分享帮助了同学，升级解锁高级分享" |
| 连续使用 7 天 | "你已连续学习 7 天！开启学习分析" |
| 教师频繁导出 | "批量导出 PPT，节省备课时间" |

### 5.2 转化原则

- **绝不**强制付费看答案
- **绝不**截断讲解步骤
- **绝不**隐藏 3D 图形
- Pro 是"支持开发者"而非"购买答案"

## 七、竞争壁垒

1. **3D 可视化质量**: 专业级 Three.js 渲染，非竞品简单的 canvas 画图
2. **AI 讲解深度**: 不只是给答案，而是分步讲解
3. **教学场景覆盖**: 从学生自学到教师上课全场景
4. **免费核心体验**: 竞品难以复制的高质量免费服务
