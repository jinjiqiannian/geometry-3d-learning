## 定价策略

| 版本 | 价格 | 定位 |
|------|------|------|
| 免费版 | ¥0 | 学生日常学习 |
| Pro 会员 | ¥19/月 (年付 ¥190) | 深度学习 + 支持开发者 |
| 教师版 | ¥29/月 (年付 ¥290) | 课堂教学工具 |
| 学校版 | 联系报价 | 全校部署 |

## 风险与应对

| 风险 | 应对 |
|------|------|
| AI 答案质量不稳定 | 人工审核 + 用户反馈 + 模型微调 |
| 竞品抄袭 | 3D 可视化技术壁垒 + 品牌社区 |
| 教育政策变化 | 定位为"学习工具"而非"培训机构" |
| 付费转化低 | 持续优化免费体验，扩大用户基数 |
| Stripe 国内支付限制 | 支付宝/微信支付集成（Stripe 已支持支付宝） |
